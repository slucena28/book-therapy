# 🧠📚 MoodRead  
### Encontre o livro certo para o seu estado emocional

> “Nem todo mundo procura um livro. Às vezes, a gente só quer se sentir compreendido.”

---

## ✨ Sobre o projeto

O **MoodRead** é uma aplicação web que recomenda livros com base no estado emocional do usuário.

A ideia nasce da conexão entre **tecnologia, psicologia e literatura**, oferecendo uma experiência mais humana e personalizada na escolha de leituras.

Em vez de buscar apenas “livros populares”, o usuário escolhe **como está se sentindo**, e o sistema sugere leituras alinhadas a essa emoção.

---

## 🎯 Objetivo

Criar uma solução que una:
- 💻 Desenvolvimento Full Stack  
- 🧠 Compreensão do comportamento humano  
- 📚 Curadoria de livros  

---

## 🚀 Funcionalidades

- Seleção de humor (triste, ansiosa, feliz, terror)
- Recomendação de livros baseada na emoção
- Exibição de título e descrição
- Integração entre front-end e back-end

---
